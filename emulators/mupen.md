## Mupen Standalone Keys
| Button             | Action             |
|:-------------------|:-------------------|
| Menu (F) + DPAD Right (Hold) | Fast Forward |
| Menu (F) + Y | Save State |
| Menu (F) + X | Load State |
| Menu (F) + DPAD Up | Cycle Save State Slot |
| Menu (F) + A | Restart Game |
| Menu (F) + Start | Quit Mupen64 |
